## Packages
leaflet | Interactive maps for hospital locations
react-leaflet | React components for Leaflet maps
framer-motion | Smooth animations for UI elements
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes

## Notes
The app requires Leaflet CSS to be imported globally.
Geolocation API will be used to get user's position for hospital search.
The backend API contracts are defined in shared/routes.ts.
